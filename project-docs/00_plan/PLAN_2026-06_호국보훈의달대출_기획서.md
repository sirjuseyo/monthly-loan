### 7-5. gift-box 이미지 상세 가이드

#### 7-5-1. 위치 및 크기 (CSS — ⚠️ 데스크탑 절대 변경 금지 / 모바일은 §7-5-4 보정 참고)

```css
/* 데스크탑 기본 — 절대 변경 금지 */
.gift-box{
  position: absolute;
  top: -110px;        /* 메인 카드 위로 110px 튀어나옴 */
  right: -55px;       /* 우측 가장자리 밖으로 55px */
  width: min(341px, 90vw);  /* 데스크탑 341px, 모바일 90vw */
  pointer-events: none;     /* 클릭 X, 순수 장식 */
  z-index: 5;               /* 메인 카드 위에 떠 있음 */
}

/* 🆕 모바일 보정 (6월 한정, T-017) — 세로 긴 이미지 텍스트 침범 방지 */
@media(max-width:767px){
  .gift-box{ width: min(250px, 68vw); top: -125px; right: -25px; }
}
```

**HTML 위치:**
```html
<div class="cards">
  <img class="gift-box" src="./2026-06/assets/[6월용].png" alt="">
  <div class="main-card">...</div>
  <div class="checker-card">...</div>
</div>
```

#### 7-5-2. 5월 페이지 와이어프레임 (gift-box 위치)

`.cards` 컨테이너 내부 DOM 구조:

```
<div class="cards"> ← position:relative 컨테이너
 │
 ├─ <img class="gift-box">       ← position:absolute, z-index:5
 │      ┌────────────────────────┐
 │      │ top: -110px            │  ← .cards 컨테이너 위로 튀어나옴
 │      │ right: -55px           │  ← 우측 가장자리 밖으로
 │      │ width: min(341px, 90vw)│  ← 데스크탑 341px / 모바일 90vw
 │      │ z-index: 5             │  ← 메인 카드 위에 떠 있음
 │      │ pointer-events: none   │  ← 클릭 X (순수 장식)
 │      └────────────────────────┘
 │
 ├─ <div class="main-card">      ← 메인 카드 (--purple 테두리)
 │      ├ .main-tag    "🌹 이달의 대출"
 │      ├ .main-title  "5️⃣🈷️은 ❤️가정의 달"
 │      ├ .main-sub    "이벤트 · 웨이팅 · 롸잇나우"
 │      └ .main-btn    [신청하기 →]
 │
 └─ <div class="checker-card">   ← 검사기 카드 (--mint 테두리)
        ├ .checker-icon  🔍
        ├ .checker-name  "대출 가능성 검사기"
        ├ .checker-desc  "내가 받을 수 있는지 확인해봐."
        └ .checker-btn   [검사 시작하기 →]
```

**시각적 위치 관계:**

```
                              ┌──────────┐
                              │   🖼️    │ ← gift-box
                          ┌───┘ gift-box │   메인 카드 우상단
                          │              │   모서리에 걸쳐서 떠 있음
  ┌───────────────────────┼──────────────┘
  │ main-card             │
  │  🌹 이달의 대출        │
  │                       │
  │  5️⃣🈷️은 ❤️가정의 달  │
  │                       │
  │  [ 신청하기 →  ]       │
  └───────────────────────┘
  
  ┌───────────────────────┐
  │ checker-card          │
  │  🔍 대출 가능성 검사기 │
  │  [ 검사 시작하기 → ]   │
  └───────────────────────┘
```

**핵심 포인트:**
- gift-box는 `.cards` 컨테이너 안에서 **첫 번째 자식**으로 위치 (메인 카드 앞)
- `position:absolute`로 띄워서 메인 카드 **우상단 모서리에 걸침**
- `pointer-events:none`이므로 메인 카드의 클릭 영역을 가리지 않음
- 데스크탑/모바일 모두 동일 위치 (반응형 width만 변동)

**6월 적용 시:**
- HTML 위치, CSS 속성, 크기 모두 **그대로 유지**
- `<img>` 태그의 `src` 속성만 6월 이미지로 교체
- alt 텍스트는 비워두는 5월 패턴 유지 (`alt=""`) 또는 6월 컨셉에 맞게 수정 가능

#### 7-5-4. 이미지 제작 사양

**CSS 배치 — 데스크탑 (≥768px, 5월·6월 동일)**

| 속성 | 값 |
|---|---|
| position | `absolute` |
| top | `-110px` |
| right | `-55px` |
| width | `min(341px, 90vw)` |
| z-index | `5` |
| pointer-events | `none` |

**🆕 CSS 배치 — 모바일 (≤767px, 6월 보정 / 2026-06-02 추가)**

> ⚠️ **6월 한정 모바일 보정 (T-017).** 6월 총 이미지(Battlefield Cross)는 비율 `1.5:1`(1536×1024)로, 5월 박스장미 `1.83:1`(2816×1536)보다 **세로가 길다**. 동일 `width`에서도 세로로 더 늘어져 모바일에서 "🪖 호국보훈의 달" 타이틀을 침범 → 모바일에서만 축소+상향 보정. **데스크탑은 불변.**

```css
@media(max-width:767px){
  .gift-box{
    width: min(250px, 68vw);  /* 축소: 90vw → 68vw */
    top: -125px;              /* 상향: -110px → -125px */
    right: -25px;             /* -55px → -25px */
  }
}
```

| 속성 | 데스크탑(≥768px) | 모바일(≤767px) |
|---|---|---|
| width | `min(341px, 90vw)` | `min(250px, 68vw)` |
| top | `-110px` | `-125px` |
| right | `-55px` | `-25px` |

> 💡 검증: 모바일 360px 기준 침범 38px → 타이틀 여백 26px 확보. 웹/앱 실기기 정상 확인 (2026-06-02). 넓은 화면(PC 최대화)에선 데스크탑 341px 유지.
> 📌 **다음 달 교훈:** 새 gift-box 이미지가 1.5:1처럼 세로가 길면 이 모바일 보정이 필요. 1.83:1 와이드 비율이면 불필요.

> 🚨 **gift-box 이미지 포맷 절대 규칙 (후배 AI 필독)**
> - gift-box 이미지는 **반드시 `.png` (투명 배경 / alpha 채널)** 로만 사용한다.
> - **절대 `.jpg`로 변환 금지!** JPG는 alpha 채널을 지원하지 않아 투명 배경이 흰색으로 채워짐.

> - 5월 검증: `Gift-Box-Rose_May-Loan.png` → **PNG 사용** (4.3MB 원본 그대로)
> - 6월 확정: `Battlefield-Cross_Jun-Loan.png` → **PNG 사용** (투명 배경 컷아웃)
> - 이미지 앱에서 열었을 때 배경이 **검정 또는 격자무늬**로 보이면 투명 배경 PNG가 맞다.

### 액션 타입별 코드 템플릿

**① 외부 브라우저(웹페이지) 연결 — `OpenWebPage`**
```json
{
  "image": "https://gitlab.com/albit-public/public/-/raw/master/banners/images/파일명.png",
  "action": "OpenWebPage",
  "service": "sir",
  "filter": ["SIR"],
  "data": "https://연결할-외부-링크"
}
```

**② 공지사항 연결 — `OpenNotice`**
```json
{
  "image": "https://gitlab.com/albit-public/public/-/raw/master/banners/images/파일명.png",
  "action": "OpenNotice",
  "service": "sir",
  "filter": ["SIR"],
  "data": 112
}
```
> `data` 값 = 공지사항 게시 번호 (숫자)

**③ 오퍼월 연결 — `OpenOfferwall`**
```json
{
  "image": "https://gitlab.com/albit-public/public/raw/master/banners/images/파일명.png",
  "action": "OpenOfferwall",
  "service": null,
  "filter": ["SIR"],
  "data": "TheZoomStore"
}
```
> `data` 값 = 오퍼월명 (문자열)
