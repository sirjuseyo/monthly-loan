### 1-3. 탈리 수식 (조건부 로직) 전체 — 스크린샷 직접 대조

#### [Page 2] 소득/지출 순차 체인 표시 (탈리 수식 직접 확인)

```
Rule 1: 연소득(1) empty → 2(월최고소득) 숨김
Rule 2: 연소득(1) not empty → 2 표시 + 3(월고정소득), 4(월최고지출) 숨김

Rule 3: 2(월최고소득) not empty → 3 표시 + 4 숨김
Rule 4: 3(월고정소득) not empty → 4 표시 + 5(월고정지출), 6(신용카드) 숨김
Rule 5: 4(월최고지출) not empty → 5 표시 + 6, 6-1 숨김
Rule 6: 5(월고정지출) not empty → 6 표시 + 6-1, 6-2 숨김
```

#### [Page 2] 신용카드 분기 (탈리 수식 직접 확인)

```
Rule 7:  6(신용카드) = "네"
           → 6-1(신용카드 월평균) 표시
           → 7(현금서비스) 숨김

Rule 8:  6(신용카드) = "아니오, 체크카드만 사용해요."
           → 6-2(체크카드 월평균) 표시
           → 7(현금서비스) 숨김

Rule 9:  6-1 OR 6-2 not empty
           → 7(현금서비스) 표시
           → 7-1(현금서비스 금액) 숨김
```

#### [Page 2] 현금서비스 분기 (탈리 수식 직접 확인)

```
Rule 10: 7(현금서비스) = "네, 지금 이용중이에요."
           → 7-1(현금서비스 금액) 표시
           → 8(소액결제), 8-1(소액결제금액) 숨김

Rule 11: 7-1(현금서비스 금액) is Value (입력됨)
           → 8(소액결제) 표시
           → 8-1 숨김

Rule 12: 7(현금서비스) = "아니오, 안쓰고 있어요."
           → 8(소액결제) 표시
           → 8-1 숨김
```

#### [Page 2] 소액결제 분기 (탈리 수식 직접 확인)

```
Rule 13: 8(소액결제) = "네"
           → 8-1(소액결제 금액) 표시

Rule 14: 8(소액결제) = "아니오"
           → 8-1 숨김

✅ 8-2(휴대폰 요금): 8번 소액결제 여부와 무관하게 항상 필수 표시
           → 탈리 제작자 확정 (2026-04-28)
           → 소액결제 네/아니오 어느 답변에도 8-2는 무조건 노출
```

#### [Page 3] 신용점수 → 대출여부 → 제도권 분기 (탈리 수식 직접 확인)

```
Rule 15: 1(NICE) empty OR 2(KCB) empty
           → 3(대출여부), 4(총잔액) 숨김

Rule 16: 1(NICE) not empty OR 2(KCB) not empty
           → 3(대출여부) 표시
           → 4(총잔액), 5(기관상세) 숨김

Rule 17: 3(대출여부) = "네"
           → 4(총잔액), 5(기관상세) 표시
           → 6(제도권) 숨김

Rule 18: 3(대출여부) = "아니오"
           → 4(총잔액), 5(기관상세) 숨김
           → 6(제도권) 표시

Rule 19: 4(총잔액) not empty
           → 6(제도권) 표시
```

> ℹ️ **신용점수 조건 해석 주의**: Rule 15 (empty OR) vs Rule 16 (not empty OR) — 하나만 입력 시 두 Rule이 동시 적용될 수 있음. 탈리는 나중 Rule 우선이므로 실질적으로 "NICE OR KCB 중 하나라도 입력하면 3번 표시"로 동작할 가능성 높음. HTML 구현 시 동일 로직 적용.

#### [Page 4] 채무변제 증빙서류 조건부 표시 (탈리 수식 직접 확인)

```
Rule 20: 6(제도권) is any of [개인회생, 워크아웃, 새출발기금, 채무조정]
           → 채무변제 증빙서류 섹션(3번 파일 업로드) 표시

Rule 21: 6(제도권) = "해당 안해요."
           → 채무변제 증빙서류 섹션 숨김
```

> ⚠️ **주의**: 신용회복위원회 선택 시 채무변제 섹션 표시 여부가 탈리 수식에서 명시적으로 확인 안 됨. Rule 20의 "any of" 목록에 신용회복위원회가 없으므로 선택 시 숨김 처리로 추정.
