## 7. 월별 대출 홈(루트 랜더) 페이지 5월 적용 기록

> 📝 본 섹션은 5월 작업 종료 후 누락된 정보를 보강한 **아카이브 기록**.  
> 5월 시점에 4월 → 5월 전환 작업 시 실제로 적용된 상태를 남겨두는 용도 (액션 아이템 아님).

### 7-1. 페이지 역할
월별 대출 홈(루트 랜더) 페이지는 매월 진입 카드만 교체되는 **고정 브랜드 페이지**. 컬러·레이아웃·구조는 절대 변경하지 않고, "이달의 대출" 카드 내용만 교체. 월별 컬러 컨셉(#3)은 **상세 페이지에서만** 적용되며, 홈 페이지는 시즌 무관 고정.

### 7-2. 고정 브랜드 컬러 ⚠️ 변경 없음

### 7-5. gift-box 이미지 위치 가이드

#### 7-5-1. CSS 위치 및 크기 (고정 — 매월 동일)

```css
.gift-box{
  position: absolute;
  top: -110px;        /* .cards 컨테이너 위로 110px 튀어나옴 */
  right: -55px;       /* 우측 가장자리 밖으로 55px */
  width: min(341px, 90vw);  /* 데스크탑 341px / 모바일 90vw */

  z-index: 5;               /* 메인 카드 위에 떠 있음 */
}
```

### 7-6. 검사기 카드 (변경 없음)
대출 가능성 검사기 카드(`.checker-card`)는 4월과 5월 모두 동일 유지. 본 페이지에서 매월 교체되지 않는 영구 요소.

### 7-7. 이 이미지 스타일 이름
5월 gift-box처럼 "배경이 투명한 PNG로, 페이지 배경 위에 자연스럽게 떠 있는 오브젝트" 스타일은 여러 이름으로 불려요:
| 용어 | 설명사용 | 맥락 |
|---|---|---| 
| 컷아웃(Cutout) | 배경에서 잘라낸 오브젝트 | 가장 일반적 |
| 투명 PNG / Transparent PNG | 기술적 명칭 (alpha channel) | 파일 포맷 관점 |
| 플로팅 오브젝트 (Floating Object) | 페이지 위에 떠 있는 디자인 요소 | 웹디자인 |
| 패키지샷 (Packshot) | 제품을 배경 없이 단독 촬영 | 전자상거래·광고
| 히어로 컷 | 아웃 (Hero Cutout)메인 비주얼로 쓰이는 컷아웃 | 마케팅 |
| 아이솔레이티드 오브젝트 (Isolated Object) | 격리된 오브젝트 사진 | 스톡 사이트 검색어 |

💡 검색할 때 가장 잘 통하는 키워드: 
"transparent background", "PNG cutout", "isolated object", "floating product"
