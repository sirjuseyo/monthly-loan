## ⑤ 이미지

- [ ] 사장님 직접 제작 → 상단 광고 배너 (`assets/` 저장)
- [ ] 이미지 용량 압축 (PNG → JPG, 1200px, **200~300KB 목표**)
- [ ] 도입부 이미지 교체 (`assets/` 상대경로 사용)
- [ ] 중간부 이미지 교체 (`assets/` 상대경로 사용)

> ⚠️ **Unsplash 외부 링크보다 `assets/` 로컬 파일 + 상대경로가 더 안정적입니다.**
> 외부 링크는 언제든 깨질 수 있어요.

### 이미지 압축 방법 (Python)

```python
from PIL import Image

img = Image.open("원본.png").convert("RGB")
# 가로 1200px로 리사이즈
ratio = 1200 / img.width
img = img.resize((1200, int(img.height * ratio)), Image.LANCZOS)
img.save("압축.jpg", "JPEG", quality=82, optimize=True)
```
