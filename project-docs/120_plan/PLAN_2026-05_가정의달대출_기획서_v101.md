## 9. 써주세요. 앱 배너 등록

### 배너 등록 (이미지 업로드)

1. GitLab 로그인
2. Project > [albit-public](https://gitlab.com/albit-public) > [public](https://gitlab.com/albit-public/public) > `banners` > `images` 접속
3. 이미지 사이즈: **1780 × 440px**
4. 파일 형식: **PNG**
5. `[+]` 아이콘 클릭 > **Upload files**
6. 배너 이미지 업로드
7. 업로드 완료 후, 해당 이미지에서 **우클릭 > 새 탭에서 열기 > URL 복사**

---

### 배너 연결 (JSON 편집)

1. Project > [albit-public](https://gitlab.com/albit-public) > [public](https://gitlab.com/albit-public/public) > `banners` > `json` 접속
2. **[Edit]** 클릭 → 편집 모드 진입
3. 배너 클릭 시 이동 목적에 따라 아래 템플릿 선택해서 추가

### 액션 타입별 코드 템플릿

**① 외부 브라우저(웹페이지) 연결 — `OpenWebPage`**
```json
{
  "image": "https://gitlab.com/albit-public/public/-/raw/master/banners/images/파일명.png",
  "action": "OpenWebPage",
  "service": "sir",
  "filter": ["SIR"],
  "data": "https://연결할-외부-링크"
}
```

**② 공지사항 연결 — `OpenNotice`**
```json
{
  "image": "https://gitlab.com/albit-public/public/-/raw/master/banners/images/파일명.png",
  "action": "OpenNotice",
  "service": "sir",
  "filter": ["SIR"],
  "data": 112
}
```
> `data` 값 = 공지사항 게시 번호 (숫자)

**③ 오퍼월 연결 — `OpenOfferwall`**
```json
{
  "image": "https://gitlab.com/albit-public/public/raw/master/banners/images/파일명.png",
  "action": "OpenOfferwall",
  "service": null,
  "filter": ["SIR"],
  "data": "TheZoomStore"
}
```
> `data` 값 = 오퍼월명 (문자열)
