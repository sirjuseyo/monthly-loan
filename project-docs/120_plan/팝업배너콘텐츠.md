        1. 선이자 수취, 이자 제한법 위한, 지인과 가족에게 대부대출 사실을 고지하는 불법 대부업체가 아닙니다.
        2. 포인트 담보 대출
            1. 써주세요.는 원금 중 일부와 이자 전부를 포인트로 상환하는 포인트 담보 대출 플랫폼입니다.
            2. 담보에 사용되는 포인트는 대출 신청 금액의 최소 13%입니다.
            3. 담보 포인트의 용도는 3가지입니다.
            - (1)크레딧 미션은 신용정보조회비용으로 사용되며, 승인 시 차감됩니다.
            - (2)원금 미션과 (3)이자 미션은 원리금 상환 시에 현금 대신 사용되며, 상환 시 차감됩니다.
            - 담보 포인트는 상환 전까지 마이 포인트에 보관됩니다. 단, 크레딧 미션 포인트는 승인 시에 차감됩니다.
            1. 39,000포인트를 담보로 300,000원을 대출해 드립니다.
            2. 담보 포인트는 신청 시에 제공해야 합니다.
            3. 심사를 진행하며, 심사에 따라 한도가 발행하지 않을 수도 있습니다. 
            4. 승인 후 상환 시에 270,000원만 입금하면 됩니다.
                1. 담보물 39,000포인트는 환수됩니다. 
                2. 30,000포인트는 원금 대신 사용되며, 상환 시 차감됩니다. 
                3. 3,000포인트는 이자 대신 사용되며, 상환 시 차감됩니다. 
                4. 6,000포인트는 신용정보조회비용으로 사용되며, 승인 시 차감됩니다.
            5. 한도 미발생(불승인) 시에 담보 포인트는 환전 처리됩니다.
            6. 포인트 담보 설정
                1. 한도 상향 신청, 롸잇나우 대출, 최초 30만 원과 50만 원 신청 : 담보 포인트 최소 14%
                2. 10만 원
                    1. 웨이팅 대출 : 담보 포인트 최소 2% (용도 : 크레딧 미션 2%) → 신청 시기별로 크레딧 미션 할인 혜택 있음
                    2. 이벤트 대출 : 담보 포인트 최소 4% (용도 : 크레딧 미션 3% + 이자 미션 1%)
                3. 20만 원
                    1. 웨이팅 대출 : 담보 포인트 최소 3% (용도 : 크레딧 미션 2% + 이자 미션 1%) → 신청 시기별로 크레딧 미션 할인 혜택 있음
                    2. 이벤트 대출 : 담보 포인트 최소 4% (용도 : 크레딧 미션 3% + 이자 미션 1%)
                4. 한도 미발생(불승인) 시에 담보 포인트는 환전처리 됩니다.
        3. 일반 대출 상품은 별도 공지합니다.
            1. 시기적 ・ 한시적 ・ 이벤트 성으로 담보 포인트, 즉 크레딧 미션 ・ 이자 미션 ・ 원금 미션이 없이, 현금으로 신용정보조회비용을 결제하고 원리금을 상환하는 대출 상품도 있습니다.
        4. 크레딧 미션 
            1. 크레딧 미션은 신용정보조회비용으로 사용됩니다
            2. 심사 전 사전 적립이며, 마이 포인트에 고스란히 보관되어 있다가, 승인 후 차감됩니다.
            3. 한도 미발생(불승인) 시에 크레딧 미션 포인트는 환전 처리됩니다.
            4. 신용정보조회비용의 시가는 10,000원입니다마는, 고객님들의 주머니 사정을 감안하여 당사가 일정 부분 부담하며 할인해 드립니다. 
                1. 웨이팅 ・ 롸잇나우 ・  블랙찬스 대출 :  신청 금액의 2%
                    1. 웨이팅 대출은 신청 시기별로 추가 할인 혜택 있습니다.
                2. 이벤트 대출 : 신청 금액의 3%
        5. 소통거리 미션
            1. 심사 자료를 말합니다.
            2. 신용보다는 비정형 ・ 비금융 ・ 정성 데이터를 받습니다.
            3. 특히 가족과 지인에게 대출 사실을 알리지 말아야 하는 법령을 준수하는 상태에서 가족과 지인과의 사회성을 취우선으로 판단합니다.  
        6. 모든 미션을 완료하면 승인 날 가능성은 상당히 높습니다.
            1. 신용이 중요하지 않지만 최종적으로 현재 연체 유무는 확인하며, 대안을 제시해 드립니다.
