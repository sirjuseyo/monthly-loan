## 작업 배경

1. **법적고지 JS 연결 (T-001):** 쮸티5호가 `sirjuseyoWeb`에서 제작한 `legal-shared.js`를 이 페이지에도 동일하게 연결. 기존 하드코딩된 하단 콘텐츠 제거 후 JS 자동 주입 방식으로 전환.

2. **480px 레이아웃 고정 (T-002):** 루트 `index.html`과 동일하게 PC에서도 모바일 너비(480px) 1열 고정. Tailwind CDN 오버라이드 문제로 `!important` 적용.

3. **법적고지 규정 준수 (T-003~T-007):** 법적고지 폰트 24px 기준, 본문 최대 허용 = 24 × 3/4 = **18px**. 규정 초과 항목 전체 18px 이하로 통일. 원 아이콘(빨간/파란) 과대 착시 현상도 함께 해소.
